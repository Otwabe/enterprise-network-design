# Executive Summary

## Overview

GreenFields Logistics Ltd. is a growing logistics and supply chain company headquartered in Nairobi, Kenya. As part of its digital transformation strategy, the company has engaged **InfraCore Solutions** to design a modern, secure, and scalable enterprise network infrastructure.

The proposed solution will replace the existing flat network with a structured enterprise architecture that improves performance, security, and manageability while supporting future expansion.

---

## Business Objectives

The new infrastructure aims to:

- Improve network performance
- Increase security through network segmentation
- Support centralized user authentication
- Enable secure remote access
- Improve wireless coverage
- Prepare the organization for future branch connectivity

---

## Proposed Solution

InfraCore Solutions proposes an enterprise network consisting of:

- Layer 3 core switching
- Departmental VLANs
- Enterprise firewall
- Active Directory
- DHCP and DNS services
- Secure Wi-Fi
- CCTV network
- VPN services
- Network monitoring

---

## Expected Benefits

Upon completion, GreenFields Logistics Ltd. will benefit from:

- Improved network reliability
- Better cybersecurity
- Simplified network management
- Increased scalability
- Enhanced user experience
- Business continuity through a resilient network design
